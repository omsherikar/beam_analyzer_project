Installation & Setup

Get the Enhanced Beam Analyzer running on your system.

System Requirements:
  • Python 3.7 or higher
  • 4GB RAM (8GB recommended)
  • Windows, macOS, or Linux
  • Internet connection for initial setup
Installation Steps:
  1. Download the project:
    git clone https://github.com/omsherikar/beam_analyzer_project.git cd beam_analyzer_project
  2. Create virtual environment:
    # Windows python -m venv venv venv\Scripts\activate # macOS/Linux python3 -m venv venv source venv/bin/activate
  3. Install dependencies:
    pip install -r requirements.txt
  4. Run the application:
    python beam_analyzer_enhanced.py
Note: The first run may take a few minutes to download and install all dependencies.

Data Format Requirements

Prepare your beam data in the correct format for analysis.

Required CSV Format:
Distance (m),SF (kN),BM (kN-m) 0.0,0.0,0.0 0.5,-25.0,-6.25 1.0,-50.0,-25.0 1.5,-25.0,-56.25 2.0,0.0,-100.0
Column Descriptions:
  • Distance (m): Position along the beam from 0 to beam length
  • SF (kN): Shear force values at each position
  • BM (kN-m): Bending moment values at each position
Excel Format Support:

The application also supports Excel files (.xlsx) with the same column structure.

Important: Ensure your data is properly formatted with consistent units (meters for distance, kN for forces, kN-m for moments).

Basic Analysis Workflow

Step-by-step guide to running your first beam analysis.

Step 1: Load Data
  1. Click the "Browse" button next to "Beam Data File"
  2. Select your CSV or Excel file
  3. Verify the data loads correctly in the preview
Step 2: Configure Analysis Parameters
  1. Select Material: Choose from the dropdown (e.g., Steel A36)
  2. Select Cross-Section: Choose beam type (e.g., I-beam)
  3. Enter Dimensions: Fill in section parameters
  4. Set Analysis Type: Enable desired analyses (Stress, Deflection, Fatigue)
Step 3: Set Constraints
  • Maximum Deflection: Set allowable deflection limit (mm)
  • Minimum Safety Factor: Set required safety factor
  • Fatigue Life: Set minimum required cycles
Step 4: Run Analysis
  1. Click "Run Enhanced Analysis"
  2. Wait for calculations to complete
  3. Review results in the Summary and Detailed tabs
  4. Examine visualizations in the plot tabs

AI Optimization Guide

Leverage artificial intelligence to find optimal beam designs.

Optimization Objectives:
  • Cost: Minimize material and construction costs
  • Weight: Reduce structural weight
  • Deflection: Minimize beam deflection
  • Safety Factor: Maximize safety margins
  • Fatigue Life: Maximize fatigue resistance
How to Use AI Optimization:
  1. Complete a basic analysis first
  2. Select optimization objective from dropdown
  3. Set constraints (deflection limits, safety factors)
  4. Click "Find Optimal Design"
  5. Wait for genetic algorithm to complete (1-2 minutes)
  6. Review optimization results comparison
  7. Apply optimized design if satisfactory
Optimization Process:
Genetic Algorithm:
  • Population-based search
  • Evolutionary optimization
  • Multi-generational improvement
  • Constraint handling
Fallback Method:
  • Systematic search if GA fails
  • Material and section combinations
  • Dimension scaling
  • Guaranteed solution
Time: Optimization typically takes 1-2 minutes. For complex problems, allow up to 5 minutes.

Troubleshooting

Common issues and their solutions.

Common Issues:
File Loading Errors

Problem: Cannot load beam data file

Solution: Ensure file format is CSV or Excel with correct column headers: "Distance (m)", "SF (kN)", "BM (kN-m)"

Optimization Timeout

Problem: Genetic algorithm takes too long or fails

Solution: The system will automatically use fallback optimization. Reduce constraint complexity if issues persist.

Installation Issues

Problem: Dependencies fail to install

Solution: Ensure Python 3.7+ is installed. Try upgrading pip: pip install --upgrade pip

Performance Tips
  • Close other applications to free up memory
  • Use smaller data files for initial testing
  • Start with simple cross-sections before complex ones
  • Enable only necessary analysis types
Getting Help:
  • Check the GitHub repository for latest updates
  • Review sample data files for proper formatting
  • Ensure all dependencies are correctly installed
  • Contact support through GitHub issues
